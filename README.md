Student Portal Website

Student Portal Website is a responsive web application designed to help students manage and access academic information online. It provides an interactive and user-friendly interface for viewing courses, grades, schedules, and announcements.

🚀 Features
View and manage student profile and details
Access course information and academic schedules
View grades and performance summaries
Responsive design compatible with desktops, tablets, and mobile devices
Clean UI using modern HTML, CSS, and JavaScript
💻 Tech Stack
Frontend: HTML (48.6%), CSS (27.2%), JavaScript (24.2%)
Responsive Design: Flexbox, Grid, Media Queries
Optional Enhancements: Animations, interactive elements with JS
